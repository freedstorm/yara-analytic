FREEDSTORM YARA ANALYTIC
Entwickelt von: freedstorm
Release: 2026-01-02

BESCHREIBUNG:
YARA-Scanner mit integrierter Freedstorm Algebra
zur intelligenten Score-Berechnung.

INSTALLATION:
1. ZIP entpacken
2. chmod +x yara-analytic
3. ./yara-analytic --update (um Regeln zu laden)

SICHERHEIT:
Prüfen Sie den SHA256-Hash in der beiliegenden
sha256sum.txt vor der ersten Ausführung.
